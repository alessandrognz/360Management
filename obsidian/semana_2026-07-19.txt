INFORME SEMANAL — 360Management
Semana del 14 al 19 de julio de 2026
=====================================

RESUMEN
-------
Semana de arranque del proyecto. Se ha establecido la base de la aplicación web
de gestión (sesiones, tareas, bandeja de entrada, ajustes) y se ha completado
el sistema de autenticación con una primera revisión importante del diseño.

INICIALIZACIÓN DEL PROYECTO (17 jul)
--------------------------------------
- Creación del repositorio y estructura inicial de la aplicación PHP.
- Incorporación de assets de imagen para la UI.

SISTEMA DE AUTENTICACIÓN (19 jul)
-----------------------------------
- Refactorización del flujo de auth: verificación de email y contraseña separadas.
- Los procedimientos almacenados ahora devuelven IDs y datos del usuario.
- Sesión: se almacena nombre e ID de usuario tras login/registro.
- Protección de páginas: redirección al login si no hay sesión activa.
- Handler de logout con limpieza correcta de sesión.

REDISEÑO DE UI (19 jul)
------------------------
- Se sustituye la estética Aero/Vista por un sistema de diseño moderno.
- Paleta de color: verde primario #4a6741, fondo #eef0ed.
- Biblioteca de iconos SVG/PNG añadida (~328 iconos: Edit, Search, Settings, Info…).
- Formularios de login/registro rediseñados con mejor UX y validación.
- Barra de navegación renovada con estados hover y tipografía limpia.
- Layout migrado a flexbox para mayor adaptabilidad.
- Seguridad: uso de htmlspecialchars() en la salida de datos de sesión.
- Eliminados assets legacy (fondos e iconos antiguos).

ESTADO ACTUAL
-------------
- Autenticación: funcional (login, registro, logout, protección de rutas).
- Páginas principales creadas: session, tasks, inbox, settings.
- UI: rediseñada y coherente en todas las páginas.
- Base de datos: esquema inicial en db.sql.

PRÓXIMOS PASOS (pendiente)
--------------------------
- Implementar funcionalidad real de tareas (CRUD).
- Desarrollar la bandeja de entrada.
- Completar la página de ajustes.
- Tests y validación de formularios del lado cliente.
